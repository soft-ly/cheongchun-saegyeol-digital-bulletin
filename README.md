# 🌿 청춘새결교회 디지털 주보 (Modern Korean Digital Bulletin)

청춘새결교회(Cheongchun Saegyeol Church) 성도님들을 위한 현대적이고 평온한 싱글 페이지 디지털 모바일 주보 웹 애플리케이션입니다.

본 프로젝트는 **세련된 은혜(Serene Grace)** 디자인 시스템을 바탕으로 구축되었으며, 모바일과 PC 화면 모두에 완벽히 최적화된 반응형 레이아웃을 제공합니다.

---

## ✨ 주요 기능 (Key Features)

- **📱 모바일 맞춤형 UI/UX**:
  - 화면 상단에서 로고와 브랜드 타이틀이 크고 가운데 정렬되어 투명하게 시작하다가, 스크롤을 내릴 때 상단 고정 바 형태로 부드럽게 축소되며 왼쪽으로 배치되는 **모핑 헤더(Morphing Header) 애니메이션**을 구현했습니다.
  - 모바일 기기에 최적화된 하단 탭 바 내비게이션을 지원합니다.
- **🖥️ PC 대시보드 그리드**:
  - 화면이 넓은 PC 환경에서는 상단에 중앙 정렬된 내비게이션 탭이 노출되며, 각 탭의 카드들이 바둑판 형태의 다중 열 그리드로 재배치되는 대시보드 레이아웃으로 자동 전환됩니다.
- **🙏 4가지 핵심 탭 서비스**:
  - **예배**: 오늘의 말씀 성경 봉독 구절, 정돈된 예배 순서, 그리고 지난 주 설교 유튜브 비디오 링크 카드 제공.
  - **소식**: 주간 교회 행사 공지사항, 함께 기도해야 할 환우/수험생 기도 제목, 그리고 성도 교제 사진 갤러리.
  - **헌금**: 십일조/감사/선교헌금 온라인 송금을 위한 가독성 높은 계좌 정보 카드 및 편리한 **원클릭 계좌번호 클립보드 복사 기능(토스트 알림)**.
  - **동행**: 새가족 환영 안내 및 입력 형식 자동 포맷팅(연락처 하이픈 자동 삽입)이 내장된 인터랙티브 새가족 등록 신청 폼.

---

## 🎨 디자인 시스템: 세련된 은혜 (Serene Grace)

디자인 명세(`serene_grace/DESIGN.md`)를 엄격히 준수하여 성스럽고 따뜻한 감성을 연출했습니다.

- **색상 팔레트 (Sage Green & Calm Neutrals)**:
  - **Primary**: 차분한 성장을 상징하는 세이지 그린 (`#3f6355`)
  - **Neutral Background**: 눈의 피로를 덜어주는 부드러운 옅은 민트빛 미색 (`#f7faf9`)
  - **Surface**: 콘텐츠 카드 가독성을 위한 순백색 (`#ffffff`) 및 세이지 그린의 음영 단계별 Elevational Surfaces.
- **타이포그래피 (Google Fonts)**:
  - **헤드라인**: 친근하고 현대적인 **고운 돋움 (Gowun Dodum)**
  - **본문 및 인용구**: 고전적이고 기품 있는 **고운 바탕 (Gowun Batang)**
- **디자인 디테일**:
  - 둥글고 부드러운 형태 감각을 지닌 Rounded Corners (카드 24px, 버튼 16px).
  - 36px의 넉넉한 세로 간격(Gap)을 배치하여 시각적 답답함을 해소하고 여백의 미를 살렸습니다.
  - 예배 탭 상단 배너가 모바일 디스플레이 가장자리까지 꽉 차도록 확장하는 Bleed 효과를 적용했습니다.

## 🚀 로컬에서 실행하는 방법

본 프로젝트는 별도의 빌드 단계가 필요 없는 순수 HTML, CSS, JavaScript로 개발되어 가볍고 빠르게 구동됩니다.

프로젝트 폴더 안에서 아래 명령어를 실행하여 웹 서버를 띄울 수 있습니다:

```bash
# Python 3을 사용하는 경우
python3 -m http.server 3000
```

서버 실행 후 브라우저에서 `http://localhost:3000`으로 접속하면 주보 페이지를 확인할 수 있습니다.

## 📂 파일 구조 (File Structure)

- `index.html` - 시맨틱 마크업 및 SEO 태그가 포함된 웹 메인 파일
- `style.css` - Serene Grace 토큰, 모바일/PC 반응형 스타일, 모핑 헤더 애니메이션 정의
- `app.js` - 내비게이션 동기화, 클립보드 복사 토스트, 새가족 폼 유효성 검사 등 인터랙션 로직
- `assets/` - 교회 공식 로고 이미지 및 갤러리 미디어 자산
- `serene_grace/DESIGN.md` - 원본 디자인 시스템 스타일 가이드라인
